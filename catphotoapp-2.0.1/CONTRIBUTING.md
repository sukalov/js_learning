# Contributing Guide

1. Fork the repo
2. Clone your fork
    ```sh
    git clone <-your fork URL->
    cd catphotoapp
    ```

3.  Branch off master
    ```sh
    git checkout master
    git checkout -b <-your branch name->
    ```
4. `npm install`
5. `gulp`

This will get you running on `localhost:8080`.

Thanks for contributing :tada:
